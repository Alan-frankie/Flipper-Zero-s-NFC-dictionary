# Instructions for use
language/EN/CN

#EN
Open your Flipper Zero software and connect it to your computer. You can download either my simplified or full version. After downloading, extract the files. You will then get two files: `mf_classic_dict.nfc` and `mf_classic_dict_nested.nfc`. Place them in the `nfc/assets` directory, and you're ready to use it!

I don't recommend using the full dictionary (because your device's RAM is limited, and using the full dictionary (114,600 entries) will easily crash your device). Therefore, I recommend using the simplified version, as it can open 95% of the cards and NFC cards on the market.
